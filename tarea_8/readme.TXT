El programa se ejecuta de manera estandar en CodeBlocks.
Al ejecutar se mostrara un menu con dos opciones.
La primera opcion dara los resultados para la ecuacion de calor
y generara archivos con datos para graficar.
La segunda opcion genera las soluciones para las matrices de 125 y 3, con 
el metodo de Jacobi y despues el metodo de Gauss-Seidel
