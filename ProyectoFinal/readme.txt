Para poder correr el archivo se debe de usar el comando:
python run.py

Si no funciona prueba con usar:
pyhton3 run.py

El archivo depende de las librerías numpy y Matplotlib
Para instalarlas debe hacer uso de pie
Usando el siguiente comando

pip install numpy
pip install Matplotlib

AL ser ejecutado el programa se podrán visualizar las imágenes y se guardaran en el dispositivo
